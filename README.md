# Digital Rok Creative

Landing page de ejemplo para una agencia de marketing digital. Incluye:

- Carrusel principal con mensajes de impacto.
- Sección de servicios con tarjetas informativas.
- Portafolio y testimonios en carrusel utilizando Swiper.js.
- Tabla de planes y formulario de contacto.

Este repositorio contiene únicamente el front-end en HTML, CSS y JavaScript.
